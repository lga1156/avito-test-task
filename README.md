### **О проекте**

Личный кабинет продавца с интегрированным AI-ассистентом, который помогает улучшать описания объявлений. Продавец видит список своих объявлений, выбирает нужное и может просмотреть карточку товара, перейти к редактированию и запросить рекомендации от нейросети. Сервис анализирует содержимое карточки товара и предлагает улучшения текста.

### **Технологический стек**

- Node.js v25.1
- React v19
- `react-router-dom` для роутинга
- TypeScript v5.9
- UI-библиотека Mantine v8
- Стейт-менеджмент — Zustand v5
- Система сборки — Vite 8
- HTTP-запросы — связка React Query + Axios
